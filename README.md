# Order Management System

A Spring Boot REST API for order management, built as a portfolio project.

## Tech Stack
Java 17, Spring Boot 3.2, Maven

## Features
- Product catalogue API
- Cart management
- Transactional checkout

## Sprints
- Sprint 1 — Domain model, OOP
- Sprint 2 — Collections, JUnit 5
- Sprint 3 — Spring Boot REST APIs, @Transactional checkout
